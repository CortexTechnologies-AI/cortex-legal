# Cortex Technologies - Legal & Compliance Pages

Static pages for SMS opt-in compliance and legal documents.

## Pages

- `index.html` - SMS opt-in form (sms.cortextechnologies.ai)
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service

## Hosting

Hosted via GitHub Pages at `sms.cortextechnologies.ai`

## Setup

1. Enable GitHub Pages in repo settings (deploy from main branch)
2. Add CNAME record in GoDaddy DNS:
   - Type: CNAME
   - Name: sms
   - Value: cortextechnologies.github.io
   - TTL: 1 hour

## Purpose

Created for Twilio toll-free number verification (+1 855-513-7511).
Twilio requires publicly accessible opt-in proof with privacy policy and terms.
